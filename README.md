# Autonomous Arduino Car with Object Detection and Avoidance

## Description
An IoT-based project that designs and builds a self-navigating 
Arduino car capable of detecting and avoiding obstacles in 
real-time without human guidance. The project demonstrates 
low-cost automation using Arduino technology and ultrasonic 
sensors.

## Project Aim
To design an autonomous Arduino-based car that can detect 
obstacles and make intelligent navigation decisions without 
human input.

## Features
- Real-time obstacle detection using ultrasonic sensors
- Automatic stopping and turning to avoid collisions
- Operates in various obstacle configurations
- Low-cost and scalable design
- LCD display for real-time system status feedback

## Hardware Components
| Component | Purpose |
|-----------|---------|
| Arduino Uno | Main microcontroller |
| Ultrasonic Sensor | Obstacle detection |
| Servo Motor | Steering and directional control |
| TT Motor (x4) | Wheel movement |
| Motor Driver | Controls motor speed and direction |
| LCD Display | Shows sensor readings and status |
| Breadboard | Circuit prototyping |
| Jumper Wires | Component connections |
| Switch | Manual power control |
| 9V Battery | Power supply |

## Software
- **Arduino IDE** - Used to write, compile and upload 
  code to the Arduino Uno microcontroller

## How It Works
1. Ultrasonic sensor continuously scans for obstacles
2. If an obstacle is detected, the car stops
3. Servo motor rotates sensor to check left and right
4. Car turns toward the clearer path
5. Car continues moving forward

## How to Run
1. Assemble hardware components as per system architecture
2. Install Arduino IDE from arduino.cc
3. Upload the obstacle avoidance code to Arduino Uno
4. Power on using the switch
5. Place car on flat surface and observe autonomous navigation

## Expected Outcomes
- Successful real-time obstacle detection and avoidance
- Smooth autonomous navigation in various environments
- Demonstration of IoT and embedded systems concepts
